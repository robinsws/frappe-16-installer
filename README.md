# frappe-16-installer
Install frappe 16 including all dependencies for Ubuntu 24.04 or Debian 13
Curl&Bash must be Installed

Ubuntu:
bash &lt;(curl -s https://raw.githubusercontent.com/robinsws/frappe-16-installer/refs/heads/main/install_ubuntu.sh)

Debian:
bash &lt;(curl -s https://raw.githubusercontent.com/robinsws/frappe-16-installer/refs/heads/main/install_debian.sh)
